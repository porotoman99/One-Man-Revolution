add(.w/.b)

add x,y will add together the values of x and y, and then store the result in y.

01  mov #0001h,&0200h  ;the value 0001h is moved into memory location 0x0200
02  mov #0003h,&0202h  ;the value 0003h is moved into memory location 0x0202
03  add &0200h,&0202h  ;the value stored in 0x0200 (0001h) and the value stored in 0x0202 (0003h) are added together (0004h) and stored in 0x0202
